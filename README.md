# Content Farm Automation

## Проектная цель

Данный проект нацелен на автоматизацию процесса создания контента для различных платформ, включая Twitter, Reddit, TikTok, и применение современных AI моделей для анализа тенденций и генерации контента.

## Функции проекта
- Интеграция с несколькими API языка программирования.
- Автоматизированные рабочие процессы на платформе Make.com.
- Генерация контента с использованием Claude.

## Быстрый старт
1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/4illi-ko/content-farm-automation.git
   ```
2. Установите необходимые зависимости:
   ```bash
   npm install
   ```
3. Настройте API ключи в `.env` файле.
   ```plaintext
   TWITTER_API_KEY=your_key_here
   REDDIT_API_KEY=your_key_here
   TikTok_API_KEY=your_key_here
   ```
4. Запустите проект:
   ```bash
   npm start
   ```
